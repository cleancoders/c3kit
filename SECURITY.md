# Security Policy

c3kit is a family of independently-versioned libraries published to Clojars
under `com.cleancoders.c3kit`. Each library has its own supported-version
table in its repository's `SECURITY.md`. The most recent published release of
each library on Clojars receives security fixes.

| Library  | Current major | Repo                                                                       |
|----------|---------------|----------------------------------------------------------------------------|
| apron    | 2.x           | [c3kit-apron/SECURITY.md](https://github.com/cleancoders/c3kit-apron/blob/master/SECURITY.md)       |
| bucket   | 2.x           | [c3kit-bucket](https://github.com/cleancoders/c3kit-bucket)                |
| wire     | 4.x           | [c3kit-wire/SECURITY.md](https://github.com/cleancoders/c3kit-wire/blob/master/SECURITY.md)         |
| scaffold | 2.x           | [c3kit-scaffold](https://github.com/cleancoders/c3kit-scaffold)            |

## Reporting a Vulnerability

Please report security issues **privately**. Do not open a public issue.

Use either of the following:

- GitHub's "Report a vulnerability" workflow on the affected library's repo:
  - [c3kit-apron](https://github.com/cleancoders/c3kit-apron/security/advisories/new)
  - [c3kit-bucket](https://github.com/cleancoders/c3kit-bucket/security/advisories/new)
  - [c3kit-wire](https://github.com/cleancoders/c3kit-wire/security/advisories/new)
  - [c3kit-scaffold](https://github.com/cleancoders/c3kit-scaffold/security/advisories/new)
- Email **gina@cleancoders.com** with details and a minimal reproduction.

We aim to acknowledge reports within 5 business days.
